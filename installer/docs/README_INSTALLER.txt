================================================================================
                           LLM Council - Installation
================================================================================

Bienvenue dans l'assistant d'installation de LLM Council!

LLM Council est un système de délibération multi-LLM qui vous permet de 
consulter plusieurs modèles de langage simultanément et de comparer leurs
réponses.

CONTENU DE L'INSTALLATION:
- Application LLM Council avec interface web
- Python 3.11 embarqué (aucune installation Python requise)
- Toutes les dépendances Python pré-installées
- Raccourci sur le bureau et dans le menu Démarrer

CONFIGURATION REQUISE:
- Windows 10 ou Windows 11 (64-bit)
- 500 Mo d'espace disque disponible
- Connexion Internet (pour accéder aux APIs LLM)

APRÈS L'INSTALLATION:
1. Double-cliquez sur le raccourci "LLM Council" sur votre bureau
2. Le serveur démarre automatiquement
3. Votre navigateur s'ouvre sur http://localhost:8001
4. Une icône apparaît dans la zone de notification (systray)

Pour plus d'informations: https://github.com/karpathy/llm-council
