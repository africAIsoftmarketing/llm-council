============================================
  LLM COUNCIL - INSTALLATION INFORMATION
============================================

Thank you for choosing LLM Council!

LLM Council is a multi-LLM deliberation system that:
- Queries multiple AI models simultaneously
- Has models review and rank each other's responses
- Synthesizes a final answer from the council's wisdom
- Supports local models via LM Studio integration

WHAT WILL BE INSTALLED:
-----------------------
1. LLM Council Desktop Application
2. Embedded Python 3.11 Runtime
3. Required Python packages (FastAPI, etc.)
4. Web-based Configuration Dashboard

SYSTEM REQUIREMENTS:
-------------------
- Windows 10/11 (64-bit)
- 500 MB free disk space
- Internet connection (for OpenRouter API calls)
- OpenRouter API key (get free at openrouter.ai)
  OR
- LM Studio installed locally (for offline/local models)

NEW IN VERSION 2.1.0:
--------------------
- LM Studio integration for local model support
- Configure per-model LM Studio server URLs
- Test connection functionality
- Mixed mode: use OpenRouter and local models together

AFTER INSTALLATION:
------------------
1. Launch "LLM Council" from your Start Menu
2. Click "Settings" to add your OpenRouter API key
   OR configure LM Studio URLs for local models
3. Start asking questions to your AI council!

For support, visit:
https://github.com/karpathy/llm-council
